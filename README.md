## About the Project

Unmatched is a turn-based strategic board game where players control unique heroes, each with their own deck and special abilities.

## Features

- Turn-based gameplay
- Character-specific decks and abilities
- Card-based combat system
- Movement system
- Health and damage management
- Modular and extensible architecture
- Object-Oriented Design

## Technologies

- C++
- Object-Oriented Programming (OOP)
- STL
- CMake
